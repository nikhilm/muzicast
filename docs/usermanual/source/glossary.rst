Glossary
========

.. glossary::
   :sorted:

   active playlist
      The playlist to which items are added when the user selects
      them in the web interface. The active playlist is always shown at the side
      of the screen.

   playlist manager
      Part of the web interface where all of the user's playlists
      can be viewed and edited.

   playlist
      A list of tracks that can be played.

   share
      Make playlists publicly available, so all users can listen
      to them or make copies of the playlist.

   web-interface
      The ProjectName 'web site' accessed through the browser
      for viewing tracks and accessing other ProjectName features.

   user
      A unique personalized account with the ProjectName server
      which provides extra features.
